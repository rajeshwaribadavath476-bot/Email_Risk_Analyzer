# Email Risk Analyzer

A simple Python program that scans email content and identifies suspicious messages.

## Features

- Detects phishing keywords
- Identifies suspicious links
- Flags risky emails
- Provides risk alerts

## Run

```bash
python Email_Risk_Analyzer.py
```

## Sample Output

```text
Enter Email Content:
Urgent! Verify your account now. Click here: http://fake-site.com

⚠️ High Risk Email Detected
```
